## 개발 스택 
- React
- TypeScript
- SASS
## 구현 내용

- 검색바에서 검색어를 입력해 API의 데이터를 Home route에 보여줍니다.
- 즐겨찾기를 추가/해제 할수 있고 즐겨찾기 route에서 즐겨찾기한 항목을 확인할 수 있습니다.
